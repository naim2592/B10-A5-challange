Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model(DOM) is a blueprint that allows developers to access and modify the elements of a web page.

How do you select an element from the DOM?

There are several methods to select an element from the DOM . To select single element we can use getElementById() or querySelector() . To select multiple elements we can use getElementsByClassName(),getElementsByTagName(),querySelectorAll().

What is event delegation in the context of the DOM and why it is useful?

Event delegation is a efficient technique to handle event .It reduces the number of event listeners by using a single listener on a parent element .

How do you manipulate an element's attributes and styles using the DOM?

We can manipulate an element's attributes by using getAttribute(), setAttribute(), and removeAttribute(). We also can manipulate styles using style property directly modify inline styles or classList methods to add, remove classes.
